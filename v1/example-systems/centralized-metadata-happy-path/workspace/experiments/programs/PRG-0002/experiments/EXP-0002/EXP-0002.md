# Decay Over Conversation Distance

## DESIGN

Using the two compression branches established in EXP-0001, test recall quality at increasing session gaps to measure how each method degrades as conversational distance grows.

- **Branch A -- Structured compression**: Context compressed with structural preservation (entity relationships, decision points, dependency graphs), as defined in EXP-0001.
- **Branch B -- Naive truncation**: Context truncated via simple token-limit cutoff with no structural awareness, as defined in EXP-0001.

Both branches are tested at four distance intervals:

- **Distance 1**: 1 intervening conversation between original context and recall attempt
- **Distance 3**: 3 intervening conversations between original context and recall attempt
- **Distance 5**: 5 intervening conversations between original context and recall attempt
- **Distance 10**: 10 intervening conversations between original context and recall attempt

At each interval, the model is prompted to recall facts, decisions, and relationships from the original context session. Intervening conversations introduce unrelated topics to simulate realistic session drift. Each branch-distance combination constitutes a distinct run condition.

## METRICS

- **Recall accuracy at each distance interval**: Percentage of prior facts correctly referenced at distances 1, 3, 5, and 10
- **Contradiction rate at each distance interval**: Number of statements in the recall session that conflict with information from the original context, measured at each distance
- **Degradation curve slope**: Rate of quality loss per session gap, computed as the linear regression slope of recall accuracy over the four distance intervals
- **Critical failure threshold**: The conversation distance at which recall accuracy drops below 50%, identified per branch
