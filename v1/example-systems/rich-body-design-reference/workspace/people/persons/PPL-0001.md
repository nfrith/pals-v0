# Ada Torres

## BIO

Ada leads incident response for agent orchestration failures that cut across routing, policy, and execution layers.

### Focus Areas

- Cross-module failure triage
- Tool routing anomalies under retry pressure
- Decision framing when evidence is partial but action cannot wait

## WORKSTYLE

### Operating Rhythm

- Prefers a 15-minute decision cadence during active incidents.
- Switches from synchronous coordination to written updates once the blast radius is contained.

### Collaboration Notes

1. Bring the next irreversible decision, not a full verbal recap.
2. Separate observed facts from proposed interpretation.
3. Flag when a mitigation changes operator workload even if customer impact is already dropping.

## ESCALATION_PREFERENCES

- Page first for active customer impact.
- Include current severity, mitigation status, and the single blocking question.
- If evidence is conflicting, send the shortest contradictory examples rather than a long summary.
