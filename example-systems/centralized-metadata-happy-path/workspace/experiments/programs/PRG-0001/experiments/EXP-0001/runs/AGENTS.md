# EXP-0001 runs/

Individual run records as `RUN-NNNN.md`.

## Discipline

A run captures what actually happened, not what should have happened. OBSERVATIONS are evidence. Write them during or immediately after the run, not days later from memory.

`outcome` stays null until the run concludes. Setting it early biases everything that follows. Let the observations speak first.

DECISION is optional and local to this run. It's "what did we learn here" — not "what should the whole program do next."
