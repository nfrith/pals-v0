# zshrc

## PURPOSE

Primary shell configuration for all managed machines. Sources plugin manager, sets PATH hierarchy, and loads environment-specific overrides.

## NOTES

- Plugin manager switched from oh-my-zsh to zinit in 2025-11 for faster cold-start
- PATH order matters: local bins before homebrew before system
