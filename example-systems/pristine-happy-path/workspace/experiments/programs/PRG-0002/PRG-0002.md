# Structured Memory Compression

## Hypothesis

Conversational context that is structurally compressed (preserving entity relationships and decision points) retains more useful signal than naive truncation when recalled across sessions.

## Success Criteria

- Across 5+ runs, structured compression leads to measurably fewer re-asks compared to a raw-truncation baseline.
- Across 5+ runs, structured compression leads to fewer contradicted prior decisions compared to a raw-truncation baseline.
